# Screenshots

This folder contains screenshots and demo images for the Elk New Tab extension.

## Recommended Screenshots

To showcase the extension effectively, consider adding:

1. **gradient-example.png** - A beautiful gradient background showcase
2. **image-example.png** - Custom image background in action
3. **video-example.gif** - Animated demo of video background (GIF format)
4. **settings-panel.png** - The settings interface
5. **widgets-showcase.png** - Highlighting all widgets (clock, weather, search, quotes)
6. **quick-links.png** - Custom quick links feature
7. **background-cycling.gif** - Animated demo of background cycling

## Image Guidelines

- **Format**: PNG for static images, GIF for animations
- **Size**: Optimize images to keep file sizes reasonable (< 500KB each)
- **Resolution**: Use high-quality screenshots (1920x1080 or similar)
- **Content**: Ensure no personal information is visible

## Usage

Once you add screenshots here, reference them in the main README.md file:

```markdown
![Gradient Background](screenshots/gradient-example.png)
```

## Note

Screenshots are gitignored by default to keep the repository lean. Update `.gitignore` if you want to commit screenshots directly, or consider hosting them elsewhere (GitHub Issues, Imgur, etc.) and linking to them in the README.
